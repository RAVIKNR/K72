# 🚀 K72 Clone

A modern clone of the K72 website built using React and Tailwind CSS. This project focuses on responsive design, smooth animations, and a pixel-perfect UI to deliver a visually engaging user experience.

---

## ✨ Features

* ⚡ Responsive design for all devices
* 🎯 Pixel-perfect UI inspired by K72
* 🎬 Smooth animations and transitions
* 🧩 Reusable React components
* 🎨 Styled with Tailwind CSS

---

## 🛠️ Tech Stack

* React
* Tailwind CSS
* JavaScript (ES6+)

---

## 📦 Installation

```bash
git clone https://github.com/your-username/k72.git
cd k72
npm install
npm run dev
```
